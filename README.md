# EndoCraft

**Make Your D&D Sessions Unforgettable**

## Struktur
- `index.html` — Landing Page (endocraft.app)
- `scroll/index.html` — Session Scroll Tool (endocraft.app/scroll)

## Deployment
1. Einmalig: `SETUP_EINMALIG.bat` doppelklicken
2. Ab dann: `deploy.bat` doppelklicken nach jeder Änderung
