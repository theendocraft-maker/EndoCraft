Tips for character art that actually looks like them

Whichever tool you use, the same things separate a generic render from your character:

Lead with the silhouette. A heavy cloak, full plate, a hood — the outline does more work than any other detail.
Give one signature object. A notched greatsword, a cracked lute, a sealed letter. It anchors the whole image.
Name a mood. Weary, defiant, serene — tone changes a picture more than features do.
Pick a moment. "Standing in a ruined cathedral" beats a floating headshot every time.
